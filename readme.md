Ornagai is a simple and clean English-Myanmar dictionary extension for Chrome/Edge. Search words directly or use the inline popup feature on any website.

- **Instant Search:** Fast word lookup using IndexedDB.

- **Modern UI:** Clean, dark-themed interface with Myanmar Unicode support.
- **Settings:** Toggle auto-popup or use "Ctrl + Select" to search.
